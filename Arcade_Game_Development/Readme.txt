1. Install Java in your desktop
2. Download the zip file from https://github.com/itzjunayed/Arcade_Game_Development.git
3. Extract the zip file
4. Go to run folder then click Meteor Dodge.jar file OR 
   Go to Eclipse IDE and create a project then create a package which called Game. 
   After that go to src/Game folder and drag & drop all the .java file in the Game package and then drag & drop icons and images to the project.
   Then click on Main.java file in Eclipse and click on run.
5. Create an account then login to your account (you can also edit and delete your account)
6. Play the game using left and right arrow key instead of (a,s,A,S)
7. Enjoy !!!
